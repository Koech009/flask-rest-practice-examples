# Flask REST API Practice Examples

This repository contains simple REST API examples built with Flask for learning purposes.

## APIs Included

### 1. Authors API

Endpoints:

- GET /authors
- GET /authors/<author_id>
- GET /authors/<author_id>/books
- GET /authors/<author_id>/books/<book_id>

### 2. Author & Book Classes API

Demonstrates object-oriented programming with:

- Author class
- Book class
- Nested REST routes

### 3. Products API

Endpoints:

- GET /products
- GET /products/<product_id>

## Requirements

Python 3.x  
Flask

Install dependencies:

pip install -r requirements.txt

## Running the API

Example:
python authors.py
